Flashcards for JS and jQuery:

JavaScript

A programming language used to make web pages interactive

Operators

A special character (or characters) that indicates an action to be performed.

Variables

They can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

A block of code that performs an action and returns a result; optionally takes arguments

Methods

An action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Arguments provide the method with additional information to help it perform its action.

Parameters

the names listed in the function definition.

Return

Result of the function.

Chaining methods

Using concatenation, or +.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

true and false are booleans. They are not strings - they simply represent being true or false.

Undefined

(any number) / 0

NaN

0/0

Escape

The \\ is called an escape. It tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.

Alert

Opens a dialog box and returns undefined.

Comments

// at the beginning of the line will comment out the line.

jQuery

A JavaScript library that makes it easier to incorporate JavaScript on your web pages.
